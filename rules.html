Rules

1- Each player scores points by marking the initial letter of their answer related with given category in the letter circle.
2- Each answer must start with a different letter in the letter circle.
3- The time required to answer the questions is 10 seconds.
4- If not answered within 10 seconds, the player gets -1 point and it is the next player's turn.
5- Each player has the right to use the pass button 2 times to pass the question without getting a minus point.
6- Using the 2nd pass or using all the letters in the circle ends the game.
7- You can continue to game for 5 turn, your scores will be stay.
